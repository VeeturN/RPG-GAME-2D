# 2D RPG Game 🗡️

A classic 2D Role-Playing Game built from scratch using Java. This project focuses on top-down exploration, interactive environments, and classic RPG mechanics.

🎓 **Academic Project:** *This project was developed as an assignment for the GUI (Graphical User Interface) course at PJATK (Polish-Japanese Academy of Information Technology).*

## 🚀 Features
* **World Exploration**: Navigate through a 2D tile-based map.
* **Collision Detection**: Solid objects, boundaries, and interactive environment.
* **RPG Mechanics**: Character stats, interactive elements, and core role-playing features.
* **Retro Graphics**: Clean 2D rendering and pixel-art style aesthetic built with Java.
* **Dynamic Camera**: The screen follows the player character across the map.

---

## 🛠️ Requirements
To run this project, you need:
* **Java Development Kit (JDK)**: Version 11 or higher.
* Any Java-supported IDE (IntelliJ IDEA, Eclipse, VS Code) or a Terminal.
