# 🗡️ RPG-GAME-2D

> Final project for the **Programming in C and C++ (PJC)** course
> **University:** Polish-Japanese Academy of Information Technology (PJATK)

A 2D top-down RPG game written in C++. The main goal of the project was to practically apply Object-Oriented Programming (OOP) paradigms, ensure proper memory management, and create a complete, playable application.

## 🎮 About the Game
You take on the role of a hero traversing a dangerous world full of enemies. Explore the map, fight monsters, and manage your inventory to grow stronger and ultimately defeat the final boss.

## ✨ Key Features
* **Movement and Exploration:** Smooth movement across a tile-based world. Built-in collision detection system.
* **Combat System:** Melee and ranged combat mechanics for dealing damage to various enemy types controlled by simple AI.
* **Inventory Management:** Item gathering, healing potions system, and weapon swapping.

## 🛠️ Technologies
* **Language:** C++ (C++17 Standard)
* **Graphics Library:** SFML 
* **Tools:** Project configured for Visual Studio / CMake.

## 🚀 How to Run (Installation)

### Requirements
* An installed environment supporting C++ compilation (e.g., Visual Studio).
* Downloaded and configured graphics library (SFML) in linker options (if not using CMake).
